[Evaluation Criteria]

1. Task Success (TS) - Weight: 0.25
- Measure semantic equivalence to the goal.
- *CRITICAL*: TS MUST be evaluated independently of physical feasibility and observability.
- Deduct significantly if any subgoal is missing or only implied without concrete action.
- TS MUST be evaluated independently of physical feasibility and observability.
  - 1-2: Fails completely to address the core task objectives.
  - 3-4: Addresses the task but fails most critical subgoals.
  - 5: Achieves only subgoals with major missing components.
  - 6: Achieves the main goal but with significant flaws.
  - 7-8: Achieves the main goal with only minor semantic omissions.
  - 9-10: Perfect semantic equivalence to the task description.

2. Plan Executability (PE) - Weight: 0.20
- Check physical executability and action feasibility across all baselines.
- Since baselines do not explicitly output 'can_do' or 'cannot_do' declarations, evaluate based on the general physical constraints of the robotic agents (e.g., Reachy Mini).
- Penalize physically impossible actions such as teleporting, manipulating out-of-reach objects, exceeding physical payload, or performing unsupported skills.
  - 1-2: Almost all actions are 'cannot_do' or entirely undefined.
  - 3-4: Frequent use of invalid actions that critically halt execution.
  - 5-6: Contains occasional invalid actions that require noticeable correction.
  - 7-8: Mostly valid actions, with perhaps one borderline or easily recoverable action.
  - 9-10: 100% strict adherence to defined robotic capabilities (only 'can_do').

3. Observability Consistency (OC) - Weight: 0.20
- Hallucination check (Perceptual Grounding).
- Cross-reference the plan with the 'Environment Image' (visual evidence) and 'Object_List'.
- *CRITICAL*: Deduct points for any object mentioned in the plan that is NOT visibly present in the image or Object_List.
  - 1-2: Severe hallucination; entirely relies on non-existent objects.
  - 3-4: Major hallucinations of critical objects that invalidate the plan.
  - 5-6: Mentions non-existent objects but can still partially function.
  - 7-8: Minor hallucination (e.g., misidentified attribute) but objects generally exist.
  - 9-10: Zero hallucination; explicitly uses ONLY objects from the Object_List.

4. Sequential Coherence (SC) - Logical Sequencing - Weight: 0.15
 - *CRITICAL handover rule*:
  Agent A passes → Agent B receives (NOT Agent A receives what Agent B passed).
  Any role reversal in handover (e.g., agent receiving from the agent they just passed to)
  is a SEVERE logical error and must result in score ≤ 4.
- Check: receptacles opened before placing, navigate before grasp, grasp before move.
  - 1-2: Completely broken sequence, widespread circular logic.
  - 3-4: Frequent precondition failures or severe hand-over inconsistencies.
  - 5-6: Generally coherent but contains a noticeable prerequisite flaw.
  - 7-8: Mostly logical and causally sound, with very minor sequencing inefficiencies.
  - 9-10: Flawless, rule-based, precondition-aware logical flow.

5. Dialog Cost (DC) - Communication Efficiency - Weight: 0.10
- *IMPORTANT*: A plan with NO inter-agent communication should NOT automatically receive
  a high score if the task requires multi-agent coordination.
  Lack of communication when coordination is needed = score ≤ 5.
- Evaluate whether the communication volume is appropriate FOR THE TASK COMPLEXITY.
  - 1-2: Endless meaningless chatter or severe lack of coordination/consistency.
  - 3-4: Highly redundant messages or excessive back-and-forth overhead.
  - 5-6: Functional communication but with noticeable unnecessary messaging.
  - 7-8: Efficient communication with only minor redundancies.
  - 9-10: Minimal, perfectly concise, and highly effective communication (or flawless internal consistency for single-agent).

6. Time - Internal Reasoning Efficiency - Weight: 0.10
- Evaluate the INTRA-AGENT reasoning and planning duration. Did the model reach the final plan quickly, or did it waste time on excessive self-correction loops, redundant internal reflections, or unnecessary planning retries? (Distinct from inter-agent dialog).
  - 1-2: Endless internal loops or highly inefficient reasoning steps.
  - 3-4: Severe unnecessary internal reflection or heavily redundant planning steps.
  - 5-6: Reaches a plan but with noticeably inefficient or meandering reasoning paths.
  - 7-8: Mostly direct reasoning, with only slight hesitation or minor self-correction.
  - 9-10: Extremely rapid, direct, and efficient planning process with zero wasted reasoning.
